# DEEP-LEARNING
I have applied the Neural Network on the MNIST dataset.

## Multi Layer Perceptron (MLP)

ABBREVATIONS 
1.BN-> BATCH NORMALISATION
2.DP-> DROPOUTS
3.IP-> INPUT LAYER
4.OP-> OUTPUT LAYER
